Discounting: definitions

  • Future Value: \( \text{FV} = PV \times (1+r)^n \)
  • Present Value: \( PV = \dfrac{FV}{(1+r)^n} \)
  • NPV of cashflows: \( \text{NPV} = \sum_{t=0}^{N} \dfrac{C_t}{(1+r)^t} \)
Reveal each bullet as you explain: FV -> PV -> NPV. Note that r must be consistent (period/year).

Code: NPV calculation (Python)


def npv(cashflows, rate):
    """
    cashflows: list of floats, C_0, C_1, ..., C_N
    rate: discount rate per period (e.g. 0.08 for 8%)
    returns: float (NPV)
    """
    return sum(cf / (1 + rate) ** t for t, cf in enumerate(cashflows))

# Example
cashflows = [-1000, 300, 400, 500]
print(round(npv(cashflows, 0.08), 2))
